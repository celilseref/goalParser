# goalParser
Gol Yorumlayıcısı:
String komutlarını yorumlayabilen bir Gol Parser'a sahibiz komut olarak girdiğimiz G,(),(l) stringlerini alarak orjinal sırayla Gol yazdırır.
Program, "for" döngüsü kullanarak, "komut" karakter dizisindeki her bir karakteri değerlendirir. "if" koşullu ifadeleri kullanarak, "G", "()" ve "(l)" karakterlerini tanır ve sonuç stringine uygun şekilde ekler.
Son olarak, "cikti" karakter dizisi null karakteri ile sonlandırılarak, sonuç stringi ekrana yazdırılır.

Bu kod, bir string komutunu yorumlayarak sonuç olarak "Gol" stringini oluşturur. Bu tür bir işlev, örneğin futbol oyunları için canlı skorlar veya gol anonsları gibi durumlarda kullanılabilir.

Ayrıca, bu kod, bir programlama örneği olarak C dilinde string işleme ve koşullu ifadelerin kullanımını gösterir. Bu nedenle, öğrenme amaçlı olarak kullanılabilir veya benzer kod bloklarının geliştirilmesine yardımcı olabilir.
